OPEN SOURCE

The term open source as the name suggests comprises two terms, open and source. So the source is available publicly,that is open for people to use and modify. Open source is a term that is originally derived from the term open source software (OSS). Open source software is the code that is designed so that it is publicly accessible-anyone can see the code, modify it, and distribute the code as they see fit.

The essence of Open Source is the community people build by contributing to the Open-source projects, products, or initiatives that embrace and celebrate principles of open exchange, collaborative participation, rapid prototyping, transparency and community-oriented development.
